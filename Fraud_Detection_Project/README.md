# AI-Powered Fraud Detection System

This is my final internship project developed at Shris Tech.

The system analyzes transaction data and predicts fraudulent activity using machine learning and anomaly detection techniques.

## Features
- Transaction dataset processing
- Fraud classification model
- Isolation Forest anomaly detection
- Performance evaluation
- Data visualization

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib & Seaborn
